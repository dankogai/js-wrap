wrap.js
=======

Universal Wrapper Object for JavaScript

Usage
-----

### From HTML:

````html
<script src="wrap.js"></script>
````

### on node.js

````javascript
require('wrap');
````


Synopsis
--------

````javascript
var _ = Object.Wrap;

````
